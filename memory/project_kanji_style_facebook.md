Edison's Facebook feed uses the "Kanji-style branded post" (Type 8 in
`facebook-content-creator/SKILL.md`) as the DEFAULT format for AI tool spotlights, tips
posts, "HOW TO" guides, and product announcements. Reference: his friend Kanji Low's
viral Facebook style.

**Locked visual structure:**
1. Top 60% of frame: Edison holding / standing beside a large glowing 3D brand logo of
   the featured tool (Claude, NotebookLM, Gemini, etc.) — warm cinematic lighting, orange/
   gold accent glow. For "tool A + tool B" posts: two logos connected by a glowing energy
   line between them.
2. Thin horizontal divider strip.
3. Small author badge: circular headshot + "Edison Chua" + blue verified tick +
   "AI Marketing Strategist" tagline in small grey text.
4. Bottom 30%: solid dark navy (#0A1628) block. Bold oversized stacked headline in WHITE
   + YELLOW (#FFD700) for key words. Under 14 words. Examples:
   - "BREAKING: I STOPPED WASTING HOURS READING TEXTBOOKS COVER TO COVER."
   - "HOW TO MASTER CLAUDE COWORK A COMPLETE GUIDE"
   - "[TOOL] CAN NOW [capability]. USE THESE [X] PROMPTS."
5. Bottom center inside navy: small white uppercase "COMMENT FOR MORE".

**Aspect ratio:** 4:5 portrait.
**Generation:** Nano Banana Pro (nano-banana-pro) via kie.ai, face reference = primary Blotato face URL.

**Why:** This format combines a scroll-stopping hero + personal brand trust (verified
badge) + a clear engagement CTA. It is the specific format Edison asked for — not a
generic YouTube thumbnail style.

**How to apply:** For any Facebook tool/tip/how-to topic, default to Type 8 unless the
topic is pure opinion (Type 1), meme (Type 5), or celebrity news (Type 3). Always pair
Type 8 with Strategy A comment thread (full tips in comments) + pin-comment image
(Edison pointing down, "ALL X BELOW") + Google Drive PDF link stored in
`rotation-state.json`.
